Lab 4 - Pseudocoding and Problem-solving

Challenge

The idea of this lab was to practice pseudocoding, and to break tasks down into a list of instructions that we could then eventually code.

Problems

We had a lot of trouble coming up with ways to phrase our psuedocode comments for java script that were action-based and suscinct. We weren't sure of which phrases to use as we had little prior experience with coding.

Results

Task 1 - Pseudocode an everyday task

First we created a list of instructions for an everyday task: How to Make Pasta!

  1. Boil Water in a pot
  2. Add Salt to water
  3. Place noodles into boiling water
  4. Wait until noodles are soft
  5. Drain noodles in colander
  6. Heat tomato sauce in separate pot
  7. Mix finished noodles in with the sauce
  8. Season to taste with salt and pepper
  9. Eat!

Task 2 - Pseudocode a Computer Task

Next, we created a list of instructions for a computer game: Connect4

  1. Create a 6x7 grid board before each game
  2. Assign each player a color
    • One player assigned blue, one player assigned yellow
  3. Receive user’s move
    • Detect the row the user selects
    • Land their token in the next available bottom space in that row
  4. Confirm user’s move
  5. Assess possible counter-moves
  6. Select best counter-move
  7. Test for Win! Condition
    • Check for 4-in-a-row of one color (diagonally, horizontally, vertically)
